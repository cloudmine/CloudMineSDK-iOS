+ (NSString *) className

The name of this class. This method must be overriden and implemented for cross-platform reasons. Choose a name that is consistent across all the platforms you are writing this app for (i.e. if all your Objective-C classes have a two-letter prefix and your Java classes for your Android version do not, you may want to stick with the non-prefixed class name for encoding and decoding purposes so everything matches up in each version of your app).
