Detailed Description

Encodes objects and scalar values into JSON form. This is meant for simple domain model relationships and should not be considered as general purpose as NSArchiver and NSUnarchiver, which can be used easily to serialize entire user interfaces.

Other than that note, this works the exact same way as standard Cocoa object serialization and deserailization works. Just implement NSCoding on your class and respond to the two methods.
