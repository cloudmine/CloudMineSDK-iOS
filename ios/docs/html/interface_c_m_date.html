Detailed Description

This class wraps NSDate so you can use it in your CloudMine-backed objects without worrying about converting to and from timestamps on serialization and deserialization. It forwards all instance methods to the NSDate instance it wraps and it subclasses the abstract NSDate class so you can use it exactly as you would use an NSDate.
