didRegisterForRemoteNotificationsWithDeviceToken: (NSData *)  devToken 

If you implement this method in your Application Delegate (for example, to do something else with the devToken, then you MUST call super BEFORE doing anything to the devToken. We do not modify the devToken in anyway, so you should call super first to ensure we don't get a malformed token.
