(CMNullStore *) + nullStore

Detailed Description

This is an internal class that should not be used in your code. It is used to represent an invalid store state after an object has been explicitly removed from its store and before it has been added to a new one. Any method that is called on an instance (other than defaultStore and nullStore, which both return the singleton instance) will throw an exception.
